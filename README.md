# NexusTimer | 프리미엄 반응형 멀티 타이머 대시보드

NexusTimer는 PC와 모바일 기기에서 동시에 사용할 수 있는 미려하고 현대적인 멀티 타이머 웹 애플리케이션입니다. 6일 23시간의 기본 시간을 템플릿으로 제공하며, 최대 50개까지의 다양한 타이머를 마음껏 추가하고 관리할 수 있습니다. 

---

## 주요 기능

1. **PC & 모바일 완벽 대응 반응형 디자인**
   - 글래스모피즘(Glassmorphism)과 네온 그래디언트 효과가 적용된 현대적이고 세련된 다크/라이트 모드 테마를 탑재했습니다.
   - 모바일 환경에서도 터치 입력이 편리하도록 넓은 터치 영역과 직관적인 스크롤 형식이 구현되어 있습니다.

2. **완벽한 상태 유지 (Local Storage 연동)**
   - 브라우저를 닫거나 기기를 재부팅해도 타이머 진행 상황이 유지됩니다.
   - 진행 중인 타이머는 현재 시각과 만료 시각(절대 시각)을 비교해 오차 없이 백그라운드에서도 완벽히 작동합니다.

3. **최대 50개 멀티 타이머 및 카테고리 관리**
   - 업무, 게임(주간 보스 리셋 등), 일상, 주방 등 필요에 따라 타이머별 카테고리와 6가지 포인트 색상을 조합하여 한눈에 식별할 수 있습니다.
   - 최대 50개 등록 제한이 걸려 있어 안정적인 브라우저 리소스 사용을 유도합니다.

4. **자유로운 정밀 편집 (일 / 시 / 분 / 초)**
   - 각 타이머는 개별적으로 이름, 카테고리, 테마 색상, 시간을 밀리초 단위까지 정확하게 수정할 수 있습니다. 
   - 기본적으로 **6일 23시간**이 설정되어 매번 긴 시간을 수동으로 입력해야 하는 번거로움을 덜어줍니다.

5. **무손실 Web Audio API 알림음**
   - 타이머 완료 시 Web Audio API로 실시간 합성한 따뜻한 느낌의 화음 벨소리가 연주됩니다. 외부 오디오 파일 다운로드가 필요 없어 로딩 속도가 빠르고 100% 동작을 보장합니다.

6. **통합 제어 & 실시간 필터링**
   - 검색창을 통해 제목이나 카테고리로 타이머를 실시간 검색할 수 있습니다.
   - '진행 중', '일시 중지', '완료' 상태 필터 탭이 있어 한눈에 모아보기가 가능합니다.
   - '모두 시작', '모두 일시정지', '모두 초기화', '전체 삭제'와 같은 일괄 작업 도구를 지원합니다.

## 로컬 실행 방법

이 프로젝트는 Vite와 React + TypeScript로 개발되었습니다. 로컬 개발 서버를 실행하려면 아래 명령어를 사용하세요.

1. **의존성 패키지 설치**
   ```bash
   npm install
   ```

2. **로컬 개발 서버 구동**
   ```bash
   npm run dev
   ```
   *터미널에 표시되는 로컬 URL(예: http://localhost:5173)을 브라우저로 엽니다.*

3. **운영 빌드 테스트**
   ```bash
   npm run build
   ```

## GitHub Pages 배포 가이드

본 프로젝트는 GitHub Pages로의 손쉬운 배포를 위해 이미 모든 설정이 구성되어 있습니다.

### 방법: GitHub Actions를 통한 자동 배포 (권장)

이미 프로젝트 루트 폴더에 `.github/workflows/deploy.yml` 파일이 작성되어 있습니다. GitHub 리포지토리에 코드를 올린 후 아래 설정을 진행하세요.

1. GitHub 웹사이트의 생성한 프로젝트 리포지토리 페이지로 이동합니다.
2. 상단 메뉴의 **Settings** 탭을 클릭합니다.
3. 좌측 사이드바에서 **Pages** 메뉴를 선택합니다.
4. **Build and deployment** 섹션의 **Source** 드롭다운에서 **GitHub Actions**를 선택합니다.
5. 로컬 코드의 브랜치 이름을 `main`으로 하여 GitHub에 푸시(`git push origin main`)하면 자동으로 빌드 및 배포 워크플로우가 실행됩니다.
6. 배포 프로세스가 끝나면 Pages 탭 상단에 배포된 라이브 사이트 주소(`https://<user>.github.io/<repo>/`)가 표시됩니다!
